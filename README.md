# AES-GCM 文件加密工具

一个轻量级、高安全性、支持图形界面和命令行的本地文件加密工具，基于 Python 编写，采用 AES-256-GCM 加密算法，适用于个人隐私保护、文件安全存储。

---

## 功能特点

- AES-256-GCM 加密算法，具备完整的认证保护
- 支持单文件加密/解密
- 支持批量加密整个目录
- 图形界面 GUI 版本：零编程基础用户可用
- 命令行 CLI 版本：适合自动化和批处理
- 默认内置加密密码，无需手动输入
- 加密文件为 `.enc` 后缀，解密后恢复 `.dec` 文件

---

### 图形和命令行界面版（GUI与CLT）
